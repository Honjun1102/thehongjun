# 文明生存树：人文社科大一统理论

这是一部从唯一公理出发、按树状结构展开的人文社科统一解释与重建之书。

这本书里有两个层次必须分清：

- `文明生存树` 负责回答：世界到底是怎样长出来的
- `问题顺序` 负责回答：人为什么总会把这棵树解释错

前者是对象，
后者是误入对象之前最常见的入口错误。
树是体，
问题是病。

我不打算把它写成一本“观点合集”，也不打算把它写成一本“名著评论集”。
它要做的是一件更硬的事：

先立根，再立证据，再批判枝条，最后把所有现象重新收回到同一棵树上。  
而第十章往前推的“问题顺序”，不是另立中心，  
而是说明：人为什么总在走到树前之前，就先把入口走歪了。

## 总原则

- 唯一公理：生存
- 主干机制：开源与节流
- 证明要求：每一步都能回到结构、边界、重复与代价
- 写作目标：不是只证明别人错，而是让读者能据此修正自己、识别现实、改变行动

## 这本书的四层结构

### 1. 根理论

- 为什么“生存”是唯一公理
- 为什么一切价值都必须服从生存前提
- 为什么所有文明都绕不开开源与节流
- 为什么群体、英雄、制度、文化、叙事都只是枝条，不是根

### 2. 证明层

这不是独立的附录，而是贯穿全书每一章的证明方法。

- 数学：负责结构、约束、阈值、最优化
- 统计：负责重复、分布、显著性、群体规律
- 地理：负责边界条件、空间约束、扩散速度、资源限制
- 自然科学：负责能量、熵、信息、进化、生态与认知的边界证明
- 《枪炮、细菌与钢铁》：作为地理与边界条件的重要参考文献

### 3. 批判书目

先批宏观总解释，再批中观单变量，再批微观机制。

### 4. 统一重写

把英雄、群众、制度、文化、神话、历史全部重写回根理论之下。

## 写作目标

这本书不是为了把别人“批倒”就结束。
它真正要做的，是把读者从以下几种误判里带出来：

- 把现象当根
- 把枝条当树
- 把一次性故事当长期规律
- 把局部经验当总体真理

最终，读者应该得到三样东西：

- 一把尺子：知道什么是根，什么是枝
- 一个坐标系：知道自己站在什么位置
- 一套方法：知道接下来该怎么判断、怎么行动

## 如何阅读这本书

这本书可以跳读，但最好不要乱读。

最稳的读法是：

1. 先读序言，先知道这棵树为什么要被建立起来
2. 再读根理论，先把生存、开源与节流立住
3. 再读证明层，先知道这套理论不是空喊口号，而是有边界、有约束、有重复、有代价
4. 再读批判书目，看看哪些经典理论只看到了枝条
5. 最后读行动出口，把理论重新收回到现实校验和具体行动里

如果你只想先抓住这本书最核心的东西，那就先记住一句话：

**先问这棵树怎样活，再问我们到底把哪一步问错了。**
