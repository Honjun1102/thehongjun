# 他们都只看到这个世界的一部分

——从文明生存树看哲学、宗教、经济、政治、历史与文化的有机关联

## 总命题

哲学、宗教、经济、政治、历史、文化，不是六个并列世界，而是同一文明生存系统在不同层面的六种显影。

这本书的根，不是价值的全部，而是文明分析的底线：

- 根：生存
- 主干：开源 / 节流
- 向上枝：哲学、宗教
- 向外枝：经济、政治
- 沉淀枝：历史、文化

这棵树试图说明两件事：

1. 文明必须先活下来。
2. 人类永远不会只满足于活下来。

因此，哲学与宗教不是噪音，而是文明伸向意义、真理和超越的两根大枝；经济与政治不是低级层，而是文明维持资源与方向的外部器官；历史与文化不是边角料，而是系统运行久了之后留在时间里和人心里的硬壳。
